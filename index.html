MEU PRIMEIRO SITE COM VIM

Muito orgulhoso e satisfeito de ter começado a usar o VIM!

Agora, listarei alguns comandos que estou aprendendo nesse editor fantástico!


Básico

  • i => modo inserção
  • Esc => sai do modo de inserção
  • I => modo inserção no início da linha
  • A => modo inserção no final da linha
  • v => entra no modo seleção
  • u => desfaz última ação (igual a ctrl+z)
  • Ctrl r => (visual mode) desfazer o desfeito ("Ctrl + y")
  • :w => (modo normal) salva arquivo
  • /palavra => (normal mode) procura pela palavra digitada

Movimentação

  • :(número da linha) => cursor para o número da linha especificada
  • $ => ir para o final da linha
  • H => ir para o topo da tela
  • M => ir para o meio da tela
  • L => ir para o final da tela
  • 0 => ir para o início da linha
  • gg => ir para o começo do arquivo
  • G => ir para o final do arquivo
  • e => (modo normal) final da palavra
  • } => (modo normal) final do parágrafo
  • w => (normal mode) move uma palavra de cada vez pra frente
  • b => (normal mode) move uma palavra de cada vez pra trás

    Edição

  • o => (modo normal) insere linha em branco abaixo
  • O => (modo normal) insere linha em branco acima
  • y => (visual mode) copia o que estiver selecionado
  • yy => (visual/normal mode) copia linha inteira
  • d => (visual mode) deleta/recorta o que foi copiado
  • dw => (normal mode) deleta palavra
  • de => (normal mode) deleta até o final da próxima palavra
  • d$ => (normal mode) deleta até o final da linha
  • dG => (normal mode) deleta até o fim do documento
  • d} => (normal mode) deleta até o fim do parágrafo
  • di(...) => (normal mode) deleta o que estiver dentro de (aspas, parênteses). Ex.: 'di()' = deleta tudo o que estiver dentro dos parênteses.
  • dd => (visual mode) deleta/recorta linha inteira
  • c => (normal mode) "change" deleta e já coloca em em modo de inserção. Pode combinar assim como o 'd'. Ex.: ci(...), cw, etc.
  • p => (visual mode) cola o que estiver selecionado/recortado
  • aw => (visual mode) selecinar palavra onde estiver o cursor
  • Ctrl V => (modo normal)seleciona a linha inteira

Avançado

  • gnc. => após procurar a palavra com "/", usar "gn" para selecionar a palavra, "c" para editar, "Esc", "n" para encontrá-la em outros locais e "." no local selecionado para alterar
  • ggVG => seleciona tudo
  • ~(espaço) (normal mode) troca maiúscula por minúscula e vice-versa
  • 3~(espaço) (normal mode) troca maiúscula por minúscula e vice-versa de três caracteres
  • U (visual mode) coloca texto selecionado em maiúscula
  • gUU (normal mode) coloca linha atual em maiúscula
  • guu (normal mode) coloca linha atual em minúscula
  • gUe (normal mode) coloca palavara atual em maiúscula
  • gU$ (normal mode) coloca linha atual em maiúscula
  • :m (# da linha) (normal mode) move linha na posição do cursor para o número informado
  • . (normal mode) repete última ação feita.
