MEU PRIMEIRO SITE COM VIM

Muito orgulhoso e satisfeito de ter começado a usar o VIM!

Agora, listarei alguns comandos que estou aprendendo nesse editor fantástico!


  • Ctrl+y+, => Emmet completar
  • Ctrl+x Ctrl+o => autocompletar no CSS
  • :# da linha => cursor para o número da linha especificada
  • $ => final da linha
  • 0 => início da linha
  • Esc => sai do modo de inserção
  • i => modo inserção
  • I => modo inserção no início da linha
  • A => modo inserção no final da linha
  • gg => ir para o começo do arquivo
  • G => ir para o final do arquivo
  • Ctrl+n => completa com palavra já digitada
  • :w => (modo normal) salva arquivo
  • v => entra no modo seleção
  • y => copia o que estiver selecionado
  • d => recorta o que foi copiado
  • /+palavra => procura pela palavra
  • w => (normal mode) move uma palavra de cada vez pra frente
  • b => (normal mode) move uma palavra de cada vez pra trás
  • /+palavra => (normal mode) procura pela palavra digitada
  • aw => (visual mode) selecinar palavra onde estiver o cursor
  • gn => após procurar a palavra com "/", usar "gn" para selecionar a mesma palavra em outros locais e depois de selecionadas todas as palavras necessárias, pressionar "c" para digitar e trocar ao mesmo tempo as palavras selecionadas
  • aw => (visual mode) selecinar palavra onde estiver o cursor
  • Ctrl+V => (modo normal)seleciona a linha inteira
  • u => desfaz última ação (igual a ctrl+z)
  • ggVG => seleciona tudo
